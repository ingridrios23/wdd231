Individual Project Site Plan

Site Name

Urban Green Spaces Initiative

This name reflects the project's focus on promoting and improving urban parks and green areas for city residents, emphasizing sustainability and community engagement.

Optional domain availability: urbangreenspaces.org

Site Purpose

The website serves as a community hub to inform visitors about local green space projects, provide resources for park activities, events calendars, volunteer sign-ups, and tips for sustainable urban gardening.

Scenarios

  • Where can I find upcoming events related to community gardening?
  • How can I volunteer to help maintain local parks?

Color Schema

Primary Color: #2a9d8f (Teal Green) - Used for headings, buttons, and highlights to evoke nature and freshness.

Secondary Color: #264653 (Dark Blue) - Used for backgrounds and navigation bars for strong contrast and readability.

Accent Color: #e76f51 (Orange) - Used sparingly for call-to-action buttons and important notices.

Background Color: #f0f4f8 (Light Blue) - Used for main page background for a calm, clean look.

Typography

  • Montserrat (Bold) - Used for all headings (h1, h2, h3) to provide a modern, strong, and clear typographic hierarchy.
  • Open Sans - Used for all body text and paragraphs for readability and a clean, neutral look.

Testing and Validation

To ensure quality, the following checks will be performed:

  • HTML validation using W3C Validator.
  • Accessibility audit using tools like WAVE and Lighthouse.
  • Color contrast checked to meet WCAG AA standards.
  • SEO best practices: semantic HTML, meaningful meta tags, alt text for images.
  • Performance optimized by minimizing CSS and images, using efficient layouts.
